Footnotes and Citations¶

footnote

1

A footnote contains body elements, consistently indented by at least 3 spaces.

If 2 is the first footnote reference, it will show up as “[1]”. We can refer to it again as 2 and again see “[1]”. We can also refer to it as note (an ordinary internal hyperlink reference).

2(1,2)

This is the footnote labeled “note”.
